### ⚡ Live CVE Threat Feed

[![CVE Feed Preview](https://raw.githubusercontent.com/MistanKh/MyPortfolio2/main/docs/screenshot-cve.png)](https://mistankh.github.io/MyPortfolio2/#cve-feed)

The portfolio pulls **live CRITICAL/HIGH CVEs** from the [NVD API v2](https://nvd.nist.gov/developers/vulnerabilities) with:
- Real-time keyword / vendor / CVE-ID search
- Per-CVE detail modal (CVSS metrics, CWE, affected products, references)
- **⚠ EXPLOIT** badge on cards cross-referenced against [ExploitDB](https://www.exploit-db.com) (refreshed daily via GitHub Actions)
- Fallback dataset auto-updated daily at 06:00 UTC — never stale
